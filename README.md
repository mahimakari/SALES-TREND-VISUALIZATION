# Sales Trend Visualization

## Project Overview

This project analyzes Amazon product data using Python and visualizes important product insights through graphs and dashboards.

It also performs prediction using Machine Learning techniques.

The project is built using:

- Pandas
- Matplotlib
- NumPy
- Scikit-learn
- Streamlit

## Features

- Data Cleaning and Preprocessing
- Top Product Categories Analysis
- Product Ratings Distribution
- Discount Percentage Analysis
- Top Product Rating Count Analysis
- Product Distribution Visualization
- Sales Prediction using Linear Regression
- Interactive Streamlit Dashboard

## Technologies Used

- Python
- Pandas
